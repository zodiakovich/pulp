# pulp — AI MIDI Generator

A papaya● tool. Generate MIDI from text descriptions. No AI API required.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel (free)

1. Push this repo to GitHub
2. Go to vercel.com → Import → select this repo
3. Click Deploy
4. Done. Your app is live.

## Features

- 20 electronic music genres with unique generation rules
- 4 layers: Melody, Chords, Bass, Drums
- 6 melody synths, 4 chord synths, 5 bass synths, 10+ drum sounds
- Genre-aware sound selection
- Web Audio playback with reverb
- Valid MIDI file export (Format 0 + Format 1)
- Style reference tags
- Manual controls (genre, key, scale, BPM, bars)
- Prompt parser for natural language input
- Zero API dependencies — everything runs in the browser

## Tech Stack

- Next.js 14 (React)
- Tailwind CSS
- Web Audio API
- TypeScript
- No external APIs required
